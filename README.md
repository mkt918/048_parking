# 名古屋駅周辺パーキングマップ

名古屋駅周辺500m圏内の駐車場を一覧表示する地図アプリケーションです。

## 機能

- 📍 名古屋駅周辺の駐車場を地図上に表示
- 💰 時間料金・1日最大料金による並び替え
- 📏 距離による並び替え
- 🎯 地図マーカーとリストの連動選択
- 📝 情報修正提案機能（GitHub Issue経由）

## デモ

<https://mkt918.github.io/nagoya-parking-map/>

## 使い方

1. 地図上のマーカー（価格表示）をクリックして詳細を表示
2. サイドバーのリストからも選択可能
3. 並び替えボタンで条件を変更可能

## データ更新

駐車場情報の修正・追加は、「情報修正を提案」ボタンからGitHub Issueで提案できます。

## 技術スタック

- Tailwind CSS
- Leaflet.js (OpenStreetMap)
